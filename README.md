# xuexitong-homework-submit

学习通（超星）作业自动化 Skill。

**当前版本：v1.1.1**

## 能力
- 扫描作业入口、解析 doHomeWork URL
- 抓题并生成答案模板
- 暂存（save）与交卷（submit）
- 手写图答案流水线（渲染→上传→HTML 回填→暂存）

## 更新检查
脚本每次运行会到 GitHub 检查最新 `VERSION`，发现新版本时会提示使用：

```bash
clawhub update xuexitong-homework-submit
```
